# App Amphibians

Simple study project to understand how to use library Jetpack Compose with Kotlin.
Following basic training concepts by Google courses.

## Requirements

- Android Studio Arctic Fox or above
- Kotlin 1.5.0 or above
- Lib Jetpack Compose
